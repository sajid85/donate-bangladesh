Frequently Asked Questions

What is the Document Object Model ?

he Document Object Model (DOM) is more like a blueprint or map of our html file. It helps our computer to find and interact with each part of the website.

How do you select element from the DOM ?

We can select an element using .getElementById , .getElementsByClassName , .getElementsByTagName , .querySelector , .querySelectorAll

What is event delegation in the contect of the DOM and why is it useful ?

Event delegation is a technique where we can attach an event listener to a parent element instead of directly to each child element. it is useful because its Efficiency , Dynamic content and Simplified code

How do you manipulate an elements attributes and styles using the Dom ?

There are 2 ways to manipulate attributes such as .getAttribute , .setAttribute and for accessing styles we mostly use

element.style.propertyName = "new-value" to set a style property directly.

element.classList.add("class-name") to add a class to the element.

element.classList.remove("class-name") to remove a class from the element.
